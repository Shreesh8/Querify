# Querify — AI Business Intelligence Platform

Querify is an AI-powered business intelligence platform that enables users to upload CSV/XLSX datasets, query data using natural language, generate dashboards, detect anomalies, and create forecasts without writing SQL.

## Features

- Natural language querying with AI
- Interactive dashboards and visualizations
- Forecasting and anomaly detection
- CSV/XLSX upload support
- Modern glassmorphism UI
- Full-stack architecture with scalable APIs

## Tech Stack

### Frontend
